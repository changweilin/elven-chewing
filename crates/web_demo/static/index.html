引擎設定

智慧選詞會依詞頻統計挑詞；模糊智慧選詞在此之上允許部分音節與聲母前綴匹配。
允許以聲母或前幾碼猜詞；即使引擎選了智慧選詞，勾選此項也會自動升級為模糊智慧選詞。
使用方式：打完不完整的注音 (例如僅輸入聲母「ㄘ」) 後，按 ↓ (Down arrow) 觸發候選字；空白鍵與其他鍵都不會啟動模糊查詢——這是 libchewing 引擎本身的設計，桌面版與沙盒一致。
取消勾選即改為「向後加詞」，影響 Ctrl+數字 加片語時的方向。
影響選字面板顯示時是否高亮目前游標位置。
會即時將預覽、送出、候選字內容轉成簡體中文。
取消勾選則改為純粹切換英文大小寫。
桌面版會在每個文件開關時記錄與還原 中文 / 英數 模式；網頁沙盒僅作為設定保存。
桌面版用於英數模式時，把實體 Dvorak、Colemak 等鍵盤反向映射回 QWERTY；網頁沙盒因瀏覽器已接管鍵盤布局，此設定僅作保存，不會重新映射輸入。
桌面版會在切換 中文 / 英數 或 繁體 / 簡體 時顯示浮動氣球；網頁沙盒改用上方狀態列短暫提示。
