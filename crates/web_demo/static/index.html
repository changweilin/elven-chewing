引擎設定 Engine Settings

模糊注音猜詞 (聲母/前幾碼)。會自動切換為 Fuzzy 引擎。
取消勾選即「向後加詞」。影響 Ctrl+數字 加片語的方向。
影響選字面板顯示時是否高亮目前游標位置。
使用 zhconv 即時轉換預覽/送出/候選字內容。
關閉時 CapsLock 改為單純切大/小寫英文。
桌面 IME 行為:於每個文件開關時記錄/還原 中英模式。Web 沙盒僅作為設定持久化。
桌面 IME 行為:英文模式時把實體 Dvorak/Colemak... 反向映射回 QWERTY。Web 沙盒由瀏覽器接管布局,此設定僅儲存,不重新映射輸入。
桌面 IME 切中英/簡繁時的浮動氣球;Web 改用畫面上方 lang bar 短暫提示。

短按 = 跟實體鍵盤一樣，僅在按壓期間生效；長按 = 持續鎖定，再點一次即放開。
